# Tingly-Box Nightly

Nightly builds for [Tingly-Box](https://github.com/tingly-dev/tingly-box) - High-performance AI gateway CLI.

## Installation

```bash
npx tingly-box-nightly start
```

Or install globally:

```bash
npm install -g tingly-box-nightly
```
